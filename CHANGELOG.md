# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-03-23

### Added
- Initial release of Calendly MCP Server
- Free tier tools: list events, get event details, search events, check availability, get busy times, list event types, get event type details
- Premium tier tools: create one-off events, cancel events, reschedule events, scheduling stats, invitee insights
- Lemon Squeezy license key validation for premium features
- OAuth2 and Personal Access Token authentication with Calendly API
- Comprehensive test suite with mocked API responses
- Claude Desktop and Claude Code configuration examples
- Full documentation with usage examples
